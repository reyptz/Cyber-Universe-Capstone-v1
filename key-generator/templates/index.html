Types de Clés Supportés

Explorez une gamme complète de primitives cryptographiques adaptées à vos besoins en sécurité.

Clés Symétriques

AES-128/192/256 et ChaCha20-Poly1305 pour un chiffrement rapide et sécurisé.

  • AES-256 (recommandé)
  • ChaCha20-Poly1305
  • Clés de session éphémères

Clés Asymétriques

RSA, ECC, Ed25519 et X25519 pour signatures et échanges de clés post-quantiques.

  • RSA 2048/3072/4096 bits
  • ECC (P-256, P-384, P-521)
  • Ed25519/X25519 (efficace)

Certificats TLS

Certificats X.509 auto-signés ou CSR pour HTTPS sécurisé.

  • Auto-signés pour tests
  • CSR pour autorités CA
  • Export PKCS#12/PFX

Clés SSH

Authentification sans mot de passe pour accès serveur sécurisé.

  • Format OpenSSH natif
  • Passphrase optionnelle
  • Support multi-algorithmes

BIP39 (Crypto)

Mnémoniques sécurisés pour wallets décentralisés.

  • 12/24 mots conformes BIP39
  • Stockage hors-ligne obligatoire
  • Compatibilité multi-chaînes

JWT/API Tokens

Clés pour authentification API et tokens signés.

  • HS256/HS512 (symétrique)
  • RS256/ES256 (asymétrique)
  • Export JWK standard
